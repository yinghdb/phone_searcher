这是一个简单的手机搜索引擎的项目。主要分为4个部分，都是我自己一个人在一周左右的时间慢慢学习完成的。
1、WebSpider中写的是一个简单的爬虫，功能从中关村爬下基本所有智能手机的型号的图片，基本信息和评论信息。
2、XmlBuild的功能是将所有手机的基本信息，包括链接存到XML中
3、DictonaryBuild的功能是根据评论信息建立字典，用于搜索引擎的检索
4、HTTPServer的功能是建立服务器，在后端进行检索，并将检索结果反馈到页面上
由于我的Server是基于Flusk的，所以没装过的话可能跑不了。
总的来说，这个项目的工作量不算太大，但是是在一周内一个人从0到有赶出来的，所以当时确实很累。做完之后还是有点成就感的。
